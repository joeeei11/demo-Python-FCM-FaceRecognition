# demo-Python-FCM-FaceRecognition

**结合模糊 C 均值聚类（FCM）的人脸识别实验，使用本地人脸数据集。**

## 功能特性

- FCM 模糊聚类人脸特征分析
- 自定义数据集加载器（data_loader.py）
- 支持多人脸数据集（dataset/）
- 实验性代码，适合研究参考

## 快速开始

### 环境要求

- Python >= 3.9
- numpy, scikit-fuzzy, Pillow

### 安装步骤

```bash
git clone https://github.com/joeeei11/demo-Python-FCM-FaceRecognition.git
cd demo-Python-FCM-FaceRecognition
pip install numpy scikit-fuzzy Pillow
```

### 基础用法

```bash
python main.py
```
